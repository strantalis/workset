# CLI

Current commands:

```
workset new <name>
workset init [--name <name>]
workset ls
workset repo add <alias|url> [--url] [--name] [--context] [--repo-dir]
workset status [--all]

workset template ls|show|create|rm|add|remove|from-workspace|apply
```

Use `-w <workspace>` to target a workspace by name or path from anywhere.
